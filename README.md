### 3. Pig Game ðŸŽ²
A two-player dice game played until one player reaches **100 points**.

**Game rules:**
- Players take turns rolling a dice.
- Each roll adds points to the current score.
- If the dice shows **1**, the current score is lost and the turn switches to the other player.
- Players can choose to "Hold" to save their current score.
- The first player to reach **100 points** wins.

**Concepts practiced:**
- Game logic
- Player state switching
- Random number generation
- DOM updates
